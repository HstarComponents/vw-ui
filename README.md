# vw-ui
 Mobile web UI Components based on Vue2 and WeUI.

# Usage

```bash
#  install
npm i -S vw-ui
```

```js
// main.js
import Vue from 'vue';
import VwUI from 'vw-ui';

Vue.use(VwUI);
```
